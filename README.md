## What is this

TUI for self-hosting and localhost management.

Analyze your system, discover running services, deploy any GitHub repo as a Docker container, and manage everything from a terminal interface. No cloud, no vendor lock, no subscriptions.

Paste a GitHub URL → get a host score (0–100) → one-click deploy.

## Install

```bash
git clone https://github.com/iamnetuseragent/ghostprovider.git
cd ghostprovider
makepkg -si
```

That's it. Dependencies (Docker, Git) will be installed automatically if missing.

## Usage

```bash
ghostprovider
```

| Action | Description |
|---|---|
| **INITIALIZE SYSTEM SCAN** | Scan environment for tools, ports, services |
| **MANAGE ACTIVE SERVICES** | Start, stop, restart, remove containers |
| **Enter GitHub URL** | Analyze and deploy any repo |
